## Nasıl Kullanılır?

1. İhtiyacına uygun uygulama veya eklenti rehberini aç
2. **Homebrew (Önerilen)** ile kur veya **Resmi İndirme** bölümünden `.dmg` indir
3. Rehberdeki önerilen ayarları uygula
4. Sorun yaşarsan rehberdeki "Bilinen Sorunlar" bölümüne bak

Yeni rehber eklemek için [TEMPLATE.md](TEMPLATE.md) şablonunu kullan.
